{\bf Problem 1:} 

Let $R$ be a UFD and $P$ be a prime ideal containing no proper prime ideals other than $(0)$.

Let $\scrP$ be a minimal generating set for $P$ (that is, with $a \slashed{\mid}b$ for all $a,b \in \scrP$). Then each $a \in \scrP$ has a prime factorization. We know that $\gcd(\scrP)$ exists; we show that it is in $P$. 

First, let $a,b \in \scrP$. Then there are $c,d$ such that $c\gcd(a,b) = a$ and $d\gcd(a,b) = b$. Now, because $P$ is prime, that means that $\gcd(a,b) \in P$ or both $c$ and $d$ are. If both $c$ and $d$ are in $P$ and $\gcd(a,b)$ isn't, then %Show that the gcd is in the prime ideal, so that $P \supset (\gcd(\scrP))$.

But naturally, every element of $\scrP$ is a multiple of $\gcd(\scrP)$. So $P \subset (\gcd(\scrP))$.

So $P$ is principal if $P$ is a prime ideal containing no proper prime ideals other than $(0)$.

{\bf Problem 3:} 

Consider $x^4 + 3x^3+3x^2-5$ in $\Q[x]$. This polynomial is irreducible if and only if it is irreducible in $\Z[x]$.

Mod out by $11$. The polynomial is equivalent to $x^4 + 3x^3+3x^2+6$ modulo $11$. So Eisenstein's Criterion applies: the polynomial is irreducible.

Thus the polynomial is irreducible in $\Z[x]$, so the polynomial is irreducible in $\Q[x]$.
